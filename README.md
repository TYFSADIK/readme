<h2>About Me</h2>

As a dedicated Cybersecurity Analyst, my expertise lies in the detection, analysis, and response to security incidents within complex, modern IT environments. I specialize in hardening and defending Microsoft Azure and Amazon Web Services cloud infrastructures, leveraging methodologies and tools common to the Security Operations Center (SOC). My career is dedicated to the proactive identification and mitigation of threats within complex IT ecosystems, informed by continuous hands-on experimentation in my home lab, where I maintain a 21-project ecosystem and architect a 7-node bare-metal Kubernetes cluster. This practice of building, testing, and securing infrastructure from the ground up forms the foundation of my professional approach to cybersecurity.

<h2>Key Projects</h2>

<table>
<tr>
<td valign="top" width="65%">

**7-Node Bare-Metal Kubernetes Cluster**  
Architected and deployed a seven-node, bare-metal Kubernetes cluster serving as a production-grade homelab environment. Provisioned infrastructure using Terraform, enforced granular network policies with Calico, exposed services externally via MetalLB, and implemented resilient encrypted storage with Longhorn. This project demonstrates end-to-end cloud-native security, high availability design, and infrastructure automation capabilities.

**TYF-AI: Self-Hosted Large Language Model**  
Developed a self-hosted LLM with integrated web search functionality at ai.tyfsadik.org. Managed resource-intensive model deployment, secured the web application interface, and implemented API protection mechanisms. This project showcases proficiency in AI/ML security, modern application architecture, and privacy-centric solution development.

**SearXNG Private Search Engine**  
Deployed and hardened a private search engine at search.tyfsadik.org as an alternative to mainstream tracking-based search services. Implemented query sanitization to prevent injection attacks, protected user search history, and ensured secure API communications. Demonstrates commitment to privacy engineering and security-by-design principles.

**Immich: ML-Powered Photo Management**  
Configured and secured a machine-learning-powered photo management system at photo.tyfsadik.org. Managed data encryption at rest and in transit, implemented fine-grained access controls, and optimized performance for ML inference workloads. Highlights skills in deploying and securing AI-driven applications with strict privacy requirements.
